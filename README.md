# Naukri Jobs Market Analysis — Mar 2023

## Overview
Power BI dashboard analysing 23,000+ Indian job postings 
from Naukri.com (July-August 2023)

## Key Insights
- Bangalore is the #1 hiring city
- Business Analyst is the most in-demand role
- SQL is the #1 required skill
- Average salary is ₹9.77 LPA
- 26.72% jobs offer 12+ LPA

## Pages
1. Overview — Total postings, salary, city & dept breakdown
2. Roles — Top job titles & salary vs experience
3. Skills — Skill demand by department
4. Company Insights — Top hiring companies

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (measures)
- Excel (source data)

## Data Source
Naukri.com job postings — Mar 2023
